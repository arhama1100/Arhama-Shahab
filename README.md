## 🧠 About the Project  

This portfolio was built with the goal of showcasing my **skills, projects, and journey** in web development — in a sleek, interactive, and responsive way.  
It includes smooth animations, 3D visual elements, and a touch of my design aesthetic.  

### 💡 Features:
- Responsive layout (Mobile • Tablet • Desktop)
- Modern 3D-inspired hero section ✨  
- Animated typing effects & smooth scroll transitions  
- Projects section with demos & GitHub links 🔗  
- Contact form with social media integration  
- Custom-designed skill bars and hover effects
